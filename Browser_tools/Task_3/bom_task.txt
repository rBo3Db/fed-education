Задание: показать схематически, из чего состоит BOM.
Нужно сделать для каждого из API ссылку на отдельную страничку, на которой будут выводиться значения указанных ниже полей, либо кнопки с действиями.
Для каждой странички вывести сверху кнопки с навигацией. Если возможна навигация назад по истории - стрелка назад, то же самое со стрелкой вперед; также указывать каждый раз длину истории посещений.

Странички:
	1. Navigator
	В зависимости от текущего браузера вывести разную информацию:
		a. Chrome
			1. userAgent
			2. appVersion
		b. Firefox
			i. appName
			ii. cookieEnabled
		c. IE
			i. platform
			ii. geolocation coordinates
	2. Screen
	У каждого пункта вывести название, краткое объяснение своими словами и значение свойства
		a. availHeight
		b. availWidth
		c. colorDepth
		d. height
		e. pixelDepth
		f. Width
	3. Location
	У каждого пункта вывести название, значение и объяснение своими словами через всплывающее сообщение
		a. hash
		b. search
		c. hostname
		d. href
		e. Pathname
	4. Window
		a. два инпутовых поля для ввода ширины и высоты
		b. кнопка
	При клике на кнопку вывести в левом углу текущие размеры окна со скроллбаром и без (опционально кроссбраузерно)
